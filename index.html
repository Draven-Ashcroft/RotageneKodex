✨ Properties of the Genetic Code

Triplet Nature

3 bases = 1 codon. 4³ = 64 possible combinations.

Degenerate

Multiple codons code for same amino acid. Leucine has 6 codons!

Unambiguous

One codon codes for only ONE specific amino acid.

Universal

Same code in bacteria, plants, animals (nearly universal).

Non-overlapping

Each base is read only once as part of one codon.

Comma-less

No gaps between codons; continuous reading.

Non-ambiguous

No codon codes for more than one amino acid.

Wobble Hypothesis

3rd base is less critical; allows flexibility in pairing.
