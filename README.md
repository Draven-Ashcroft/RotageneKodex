# 🧬 ROTAGENE KODEX

### 🔬 Interactive Codon Dialer + Genetic Code Laboratory

An interactive biology tool that brings the **Genetic Code** to life through a 3-ring codon dialer, real-time translation, mutation simulation, and a fully explorable codon reference table.

---

## ✨ Features

🔹 3-Ring rotating codon dialer

🔹 Real-time codon decoding

🔹 mRNA to protein translator

🔹 Mutation simulator (Silent / Missense / Nonsense)

🔹 Interactive 64-codon reference table

🔹 Quick-jump to START and STOP codons

🔹 Smart panel & classroom-ready

🔹 Single file — works fully offline

---

## 🌀 The Three Rings

| Ring | Position | Bases |
|------|----------|-------|
| **Ring 1** | 1st base | U, C, A, G |
| **Ring 2** | 2nd base | U, C, A, G × 4 |
| **Ring 3** | 3rd base | U, C, A, G |

Rotate each ring independently — the dialer reads the top-aligned base from each ring and instantly decodes the codon and amino acid.

---

## ⚗️ Panels

- **Codon Decoder** — Type any 3-base codon, get the amino acid instantly. Syncs with the dialer.
- **mRNA Translator** — Enter a full mRNA sequence (AUG onward), get the complete protein chain with color-coded badges.
- **Mutation Simulator** — Load a translated sequence, click any base to mutate it, and see the biological effect in real time.
- **Codon Table** — All 64 codons, color-coded by amino acid. Click any cell to jump the dialer to that codon.

---

## 🎯 Learning Outcomes

After using Rotagene Kodex, students can:

✅ Read and decode any mRNA codon

✅ Understand codon degeneracy and wobble hypothesis

✅ Identify START (AUG) and STOP (UAA, UAG, UGA) codons

✅ Distinguish silent, missense, and nonsense mutations

✅ Relate codon changes to protein-level consequences

✅ Recall all 8 properties of the genetic code

## 🚀 Getting Started

1. Open `GENETIC_CODE.html` in any modern browser.
2. Rotate the rings or use ▲▼ controls to dial a codon.
3. Read the decoded result in the side panel.
4. Enter an mRNA sequence in the Translator panel.
5. Load it into the Mutation Simulator to explore effects.
6. Use the Codon Table to jump to any codon directly.

> No installation. No internet required. Just open and teach.
